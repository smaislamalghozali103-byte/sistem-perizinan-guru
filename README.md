# SIPEG - YPI Pondok Modern Al-Ghozali
Sistem Informasi Perizinan Guru berbasis Python (Streamlit).

## Cara Menjalankan
1. Instal pustaka yang dibutuhkan:
   ```bash
   pip install -r requirements.txt
   ```
2. Buat file `.env` dan masukkan API Key Gemini Anda:
   ```env
   GEMINI_API_KEY=kunci_rahasia_anda_di_sini
   ```
3. Jalankan aplikasi:
   ```bash
   streamlit run app.py
   ```
